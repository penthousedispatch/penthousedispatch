## Full description draft

Penthouse Dispatch is a premium transportation operations platform built for non-emergency medical transportation providers.

The app combines live dispatch, company operations, driver tools, rider tracking, AI-assisted routing, and operational testing into one system.

With Penthouse Dispatch, providers can:

- run live dispatch from a map-first dashboard
- manage company drivers and onboarding
- review marketplace and imported trips
- monitor rider tracking and trip progress
- control company AI scheduling preferences
- manage driver pay and payout operations
- test workflows safely in sandbox mode before live changes

Drivers can:

- sign in and start shifts
- receive trip instructions
- update trip statuses
- choose preferred work zones
- complete onboarding
- hear audio guide instructions if needed

Riders can:

- receive a live tracking link
- view accepted trip progress
- identify the assigned vehicle and driver
- use guide audio for instructions

Penthouse Dispatch is designed to feel modern, fast, and operationally clear, while still supporting the compliance and workflow needs of growing NEMT companies.
