# Daycare Release Policy Template

Use this template later for daycare-specific release rules.

## Core Rule

No child should be released unless the transport team can verify that the receiving adult is allowed to receive the child.

## Required Checks

- Match the authorized pickup list
- Confirm guardian or receiving adult identity
- Confirm site-specific release restrictions
- Confirm any special handoff note for the child

## Block Dispatch Or Handoff If

- authorized pickup list is missing
- release restrictions are unclear
- guardian contact cannot be reached when required
- daycare site policy requires adult handoff and no approved adult is present

## Incident Logging

If release fails, log:

- child name
- site name
- attempted receiver
- reason for block
- who was contacted
- final disposition
