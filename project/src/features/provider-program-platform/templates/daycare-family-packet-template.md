# Daycare Family Packet Template

Use this template later for daycare family intake.

## Child Profile

- Child full name
- Date of birth
- Classroom / age group
- Program or site name

## Guardian Contacts

- Primary guardian
- Secondary guardian
- Best daytime phone
- Emergency phone

## Authorized Pickup Contacts

- Name
- Phone
- Relationship
- Can receive child from transport: yes / no

## Release Restrictions

- Must be released to adult only
- May not be released to listed restricted contacts
- Special handoff instructions

## Future Ride Permission

- Parent understands child can be enrolled now for future rides even if no rides are active today
- Parent approves future ride requests for appointments, programs, sports, and special events

## Medical / Behavior / Mobility Notes

- Allergy notes
- Mobility support
- Aide or supervision needs
- Behavior / sensory considerations

## Sports / Event Transport Needs

- Football or sports schedule interest
- Program events
- School or community events
